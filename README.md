# social_app

Social Flutter application.
Feature : 
-Boarding Screen .
- Login / Sign up 
- Confirm Email Verification 
- Create post (with photo or camera shot)
- Chat room
- Add Friends 
- profile and Edit profile data
- FCM Notifications.
- Search
